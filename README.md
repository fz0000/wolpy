# WOLPY
Python 3 Wake-on-LAN script.

  * Single file program
  * No external dependencies
  * Encapsulates magic pattern into UDP Frames
  * Supports IPv6 *(use FF02::1 for local network segment)*
  * Arguments checking
  * Exceptions handled
  
Tested with python 3.5.2 (linux) and python 3.6.1 (windows)
